# raytracer
Simple raytracer with python

## Feature
- Lambertian Shading
- Phong Shading
- Area Lighting
- Environment Lighting
- Ambient Occlusion
- Mirror Reflection

## Dependency
- Python3.2
- Pygame1.9.2
